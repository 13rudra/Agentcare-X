About AgentCare X

🚀 How the System Works

AgentCare X is a simulated OpenEnv benchmark environment for evaluating autonomous AI customer support agents. It simulates dynamic customer interactions where the agent must reason about the customer's emotions, use backend tools (like checking inventory or processing refunds), and resolve issues efficiently without hallucinating.

🎮 How to Use It

  • Mock Mode: Select a task and click "Start Episode" to watch a pre-scripted agent interact with the environment automatically.
  • Manual Mode: Play as the agent yourself! Select a task, choose "Manual (You play)", and hit start. Use the "Respond" tab to chat, and the "Use Tool" tab to execute backend commands.
  • Scoring: Your performance is graded in real-time based on your tool usage accuracy, interaction efficiency, and how well you manage the customer's emotion level.
